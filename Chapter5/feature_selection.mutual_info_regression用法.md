# feature_selection.mutual_info_regression用法



```python
sklearn.feature_selection.mutual_info_classif(X, y, 
                                              discrete_features='auto', 
                                              n_neighbors=3, 
                                              copy=True, 
                                              random_state=None
                                             )
```

`X` :特征矩阵

**`y`** :目标向量

**discrete_features** : **{‘auto’, bool, array_like}, default ‘auto’**
